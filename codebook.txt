Code Book
======================
The tidyData data set consists of:
- 180s rows which correspond to the statistical average of data for 30 subjects each performing 6 activities.
- 68 columns of summary statistics for each combination of subject and activity.

NOTE: No unit information was available for the initial data file. Where possible, enumerated data has been listed explicitly. Indication of data range is provided as a Min and Max for the data set.
Floating point number are denoted [F]
Enumeration-type data is denoted [E]

[1] Activity
	 [E] Activity Description
		 laying
		 sitting
		 standing
		 walking
		 walkingDownstairs
		 walkingUpstairs 


[2] Subject
